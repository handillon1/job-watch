Create the Applications sheet
  1. Go to sheets.new to create a fresh Google Sheet.
  2. Rename a tab to exactly Applications (case-sensitive). Double-click the tab name at the bottom to rename.
  3. In row 1, paste this header across cells A1:F1: 
    Company	Role	Date Applied	Email Applied	Resource	Status
    (tab-separated — pasting will fill the cells correctly)
  4. Copy the sheet ID from your browser URL: docs.google.com/spreadsheets/d/<THIS_PART>/edit.
  5. Save that ID as the APPLICATIONS_SHEET_ID GitHub secret.
Share the sheet with the service account
  1. The service account email is shown on the GCP service account card above (looks like ...@<project>.iam.gserviceaccount.com).
  2. Open your Google Sheet → click the Share button (top right).
  3. Paste the service account email into the "Add people" field.
  4. Set permission to Editor (the bot needs to append rows).
  5. Uncheck "Notify people" (service accounts can't read email).
  6. Click Share. Refresh this dashboard — the Sheets card should turn green.
Set up the Status column dropdown
  1. Click the F column header to select the whole column.
  2. Go to Data menu → Data validation.
  3. Set Apply to range to Applications!F:F (covers all current and future rows).
  4. Criteria: Dropdown. Add these items one per row:
    • Applied (required — the bot writes this exact string)
    • OA
    • Interview
    • Offer
    • Rejected
  5. Click each item to assign a color (e.g. Offer=green, Rejected=red).
  6. Under Advanced options, choose how to handle invalid input — "Show a warning" is safer than "Reject input" since the bot might write something unexpected.
  7. Save. New cells in column F will render as colored chips.
Add a summary/stats tab

Optional: a second tab that counts applications by status. The bot writes only to A:F of the Applications tab, so this won't be touched.

  1. Click the + at the bottom-left to add a new tab. Rename it to Summary.
  2. In A1:B5, paste: 
    Applied:	=COUNTIF(Applications!F:F, "Applied")
OA:	=COUNTIF(Applications!F:F, "OA")
Interview:	=COUNTIF(Applications!F:F, "Interview")
Rejected:	=COUNTIF(Applications!F:F, "Rejected")
Offer:	=COUNTIF(Applications!F:F, "Offer")
  3. Optional row 6: total apps with =COUNTA(Applications!A:A)-1
  4. Optional row 7: response rate =(B3+B4+B5)/B6 formatted as percent
